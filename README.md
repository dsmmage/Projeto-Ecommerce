# MVPgames — E-commerce de Games

Projeto fictício de e-commerce para portfólio, desenvolvido como analista de e-commerce.

## 🎮 Sobre o Projeto

Loja online de games, consoles e acessórios com design moderno baseado nas tendências de 2026.

## ✨ Funcionalidades

- Catálogo com 12 produtos e filtros por categoria
- Carrinho com cupons de desconto (MVP40, BRONZE10, FRETE)
- Checkout em 3 etapas com cálculo de frete
- Sistema de wishlist e pedidos com rastreamento
- Chat com IA integrado
- Design com glassmorphism, bento grid e tipografia escultural
